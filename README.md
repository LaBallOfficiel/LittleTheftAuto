# LittleTheftAuto
LittleTheftAuto est un GTA-like en 2D vue du dessus où tu explores une ville rétro, fais des courses-poursuites, voles des véhicules et enchaînes des missions rapides. Le site présente l’univers, les persos, les zones, ainsi que les actus et mises à jour du jeu. Fun, chaos et liberté au menu.
